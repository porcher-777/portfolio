Giga Ligma Eshop

Description du projet

Principe

Le principe de ce projet est de construire une interface utilisateur pour un site d’e-commerce où les utilisateurs peuvent parcourir, visualiser et acheter différents produits. Le front-end est conçu avec React, ce qui permet de rendre la page dynamique et réactive, tandis que Tailwind CSS facilite la mise en page et le style moderne du site. La base de données DummyJSON est utilisée pour charger et afficher des produits fictifs (comme des meubles, des légumes, etc.) dans la boutique en ligne.

But

Le but du projet est d’enseigner et de mettre en pratique plusieurs technologies web, comme React, Tailwind CSS, et la manipulation de données via une API (DummyJSON). Le TP permet aussi de comprendre comment structurer et organiser une boutique en ligne, gérer l’affichage de produits et implémenter des fonctionnalités comme les filtres, l’ajout au panier, et la navigation sur plusieurs catégories.

Finalité

L’objectif final est de fournir une interface fonctionnelle qui permet à un utilisateur de sélectionner et acheter des produits fictifs, en simulant toutes les fonctionnalités essentielles d'un véritable site e-commerce. Cela inclut le tri des produits, l'affichage détaillé des articles, la gestion du panier, et la recherche d’articles spécifiques. À la fin, l'étudiant aura une compréhension complète du développement d'une application web d'e-commerce.
